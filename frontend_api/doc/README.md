## Введение

Platform API предоставляет программный интерфейс для взаимодействия с платформой. API построено на принципах REST и использует JSON для передачи данных.

## Базовый URL

```
{{baseUrl}}/f-api/
```

## Аутентификация

Для доступа к защищенным ресурсам API требуется аутентификация. Подробная информация о процессе аутентификации, включая регистрацию, вход в систему и управление токенами, доступна в [документации по аутентификации](auth.md).

## Формат ответов

Все ответы API возвращаются в формате JSON. Успешные ответы обычно имеют следующую структуру:

```json
{
  "success": true,
  "data": {
    // Данные ответа
  }
}
```

## Обработка ошибок

В случае ошибки API возвращает соответствующий HTTP-статус и JSON-объект с информацией об ошибке:

```json
{
  "success": false,
  "error": {
    "code": "ERROR_CODE",
    "message": "Описание ошибки"
  }
}
```
